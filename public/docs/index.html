For Admins...

Notes
  • To create a new user, assign a generic password and instruct the user to use the "password reset" functionality from the homepage. This is the only way to reset your password currently.

For Devs..

Notes
  • Themes should have a form whos action="submit.php" - The CMS includes javascript to automatically handle ajaxing data and sending emails.
  • Dashboard -> Websites -> SITE -> Email value should not have a trailing slash in the URL
  • To generate sitemap, run following from CLI: php generateSitemap.php.php https://www.example.com

Creating Blogs..
  • To add support for a "blog" to a theme, your theme must have both a "Blog List" and a "Blog Post" layout with "blog-list" and "blog-post" keys, respectively.
  • The posts can be accessed on the frontend via $cms->posts
Creating Galleries..
  • To add support for a "gallery" to a theme, your theme must have a "gallery" layout with a key = "gallery"
